Primary corrected finding

Active sub % is the most consistent pre-collapse warning — not GC posting rate

Active sub % declined in 53% of all collapse events within the 3-month window before the break. This is the strongest single verified warning signal. GC posting rate, despite being the best leading indicator for sustained high retention, only fell in 31% of collapses — it was actually rising in several major breaks including Kansas City, Virginia Beach, and Milwaukee.

Important correction to prior analysis

GC posting rate is a floor condition — not a reliable collapse predictor

The previous report overstated GC posting rate as a collapse warning. The verified data shows it falls in only 31% of collapses. However: no market with GC posting consistently above 19% suffered a major collapse. Markets drifting to 13–15% are structurally vulnerable when combined with declining active sub % — but the rate alone does not reliably signal a break.

Notable anomaly

Milwaukee's 28pp collapse had no warning from any verified metric

Milwaukee — the largest drop in the dataset — showed all three metrics flat or rising before breaking below 75% in January 2026. GC posting was rising (9% → 19%), active sub % was rising, project penetration was flat. This collapse either had an external cause not in the data, or it represents a structural break that the metrics we have cannot predict. It should not be used as an example of the warning pattern.

Compound signals — what severity looks like

When more verified metrics are declining simultaneously, collapses tend to be larger. Based on the 15 major collapse events (drop ≥8pp):

1 signal
Small break — typically 2–5pp. Often recovers within a few months. Low urgency.
2 signals
Moderate break — 5–10pp. Slower to recover. Intervention likely needed.
3 signals
Major collapse — 10–20pp. Kansas City and Atlanta both showed all three declining before break.
Caveat

No pattern predicts all collapses — Milwaukee is the counterexample

Milwaukee dropped 28pp with zero warning from any verified metric. Any monitoring framework using these signals will miss some breaks. The signals are useful for raising alert probability — they are not deterministic. Wisconsin's collapse may require additional investigation for an external cause.

04

The monitoring playbook — verified signals only

01

Watch — active sub % softening

Active sub % has declined month-over-month for 1–2 months but is still above 28%. GC posting and project penetration stable. Retention still above 78%.

Monitor monthly. No immediate action.
02

Warning — active sub % falling + one other

Active sub % declining for 2+ months AND either GC posting rate OR project penetration also declining. Retention still above 75% but compressing toward threshold.

Investigate GC posting activity. Check if subs are searching but not finding projects.
03

Critical — all three signals declining

Active sub %, GC posting rate, AND project penetration all trending negative simultaneously for 3+ months. This combination preceded Kansas City's 19pp collapse.

Escalate. GC activation campaign. Investigate sub disengagement cause.
04

Break — retention crosses below 75%

Warning window has closed. Recovery is possible (Pittsburgh: ~6 months, Louisville: ~4 months) but requires active sub % and GC posting rate to stabilize and recover first.

Recovery mode. GC activation over sub acquisition. Do not acquire into a broken market.
Timing

3 months is the minimum window — but the signals aren't always there

The pre-collapse signals typically appear 3–6 months before a break when they appear at all. However, given that Milwaukee collapsed with zero warning and other markets (Virginia Beach, Richmond) also showed no consistent pattern, a monthly monitoring cadence is essential. A quarterly review cycle would miss many warning windows entirely — and some collapses will still arrive without warning.

Markets at risk — March 2026

Applying the verified warning scorecard to the most recent 3 months of available data (Jan–Mar 2026). Only the active sub % signal is strong enough to use as a standalone indicator.

Highest risk — active sub % declining + GC posting below floor

Milwaukee (74%, GC posting 12%) and Cincinnati (71%, GC posting 13%)

Milwaukee holds at 74% but GC posting rate is 12% — well below the 19% floor threshold and the lowest of any near-threshold market. Active sub % has been volatile. Milwaukee's previous collapse in January 2026 shows it can drop sharply without warning. Cincinnati at 71% with 13% GC posting is structurally similar and already below 75%.

Watch — active sub % declining in 3mo window

Raleigh (74% in Feb 2026, declining) and Charlotte (71%, recently broke)

Raleigh's active sub % was declining in the 3 months before its February 2026 break from a 22-month streak. Charlotte broke in March 2026 after 26 months — its active sub % was flat but GC posting fell from 28% to 21% in the prior 6 months. Both now sit in the 70–74% range with declining trends.

Stable — no active warning signals from verified metrics

Detroit, Pittsburgh, Louisville, Austin — clean on all three metrics

None of the four improving or stable markets show the dual-decline pattern in recent months. Detroit has held above 78% for 26 months with GC posting consistently above 17% and active sub % stable. Pittsburgh and Louisville are recovering. Austin holds at 78% with GC posting steady at 18% and active sub % not declining.
